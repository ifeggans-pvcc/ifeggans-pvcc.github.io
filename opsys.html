Operating Systems Terminology

Operating Systems Terminology
Term Definition
Operating System The software that controls the overall operation of a computer.
Job Queue Jobs residing in mass storage wait for execution here.
FIFO(First-in,First-out) This makes sure that objects are removed from queue in the order which they arrived.
Interactive Processing A terminal consisted of little more than an electronic typewriter, by which the user could type input and read the computer's response that was printed on paper.
Time-Sharing Provides service to multiple users at the same time.
Multiprogramming Time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
Application Software Consists of the programs for performing tasks to the machine's utilization.
System Software Performs tasks that are common to computer systems in general.
Utility Software Programs for performing activities that are fundamental to computer installations.
Graphical User Interface Makes objects to be manipulated, such as files and programs are represented pictorally on the display as icons.
Shell Older user interfaces.
Kernel The internal part of an operating system.
Device Driver Software units that communicate with the controllers to carry out operations on the perpheral devices attached to the machine.
File Manager Coordinate the use of the machine's mass storage facilities.
Memory Manager Coordinates the machine's use of main memory.
Virtual Memory A large "ficitional" memory space created by paging.
Boot Strapping Transfers the operating system from mass storage into main memory.
ROM Special nonvolatile memory cells, where the initial program is constructed from.
Deadlock A condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling Holding data for output at a later but more convenient time.
Super User The person with highly priveleged access to the operating system by identifying theirself during the login procedure.
Multi-Factor Authentication When you have to require more than one piece of evidence to gain access.
