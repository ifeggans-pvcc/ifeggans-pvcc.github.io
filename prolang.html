Chapter 6: Programming Languages

Chapter 6 Terms
Term Definition
Machine Language Sequences of instructions that are encoded as numeric digits; An encoding system.
Debugging Locating and correcting errors.
Assembler A program that converts mnemonic expressions into machine language instructions.
Assembly Language A mnemonic system for representing programs.
Translator A program that translated programs expressed in high-level primitives, into machine language.
Compiler A type of translation program that compile several machine instructions into short sequences.
Interpreter Execute the instructions as they were translated instead of recording the translated version for future use.
Cross-Platform Software Software that is independent of an operating systems' design as well as the machine's hardware design.
Natural Languages Such as English, German, or Latin.
Formal Languages Programming languages
Machine Independence Third generation languages, that don't refer to the attributes of any particular machine, so they can be compiled from one machine to another.
High-level Programming Language Allow locations in main memory to be referenced by descriptive names, rather than numeric addresses.
Object-oriented Programming The functions describing how an object should react to an outside stimulus.
Scripting Languages A subset of imperative programming languages typically ised to perform administrative tasks rather than to develop complex programs.
Data Types:
  • Integer: Refers to numeric data consisting of whole numbers.
  • Float: Refers to numeric data that might contain values other than whole numbers.
  • String:One string of symbols concatenating.
Primitive Data Types Data types that are primitives in a programming language. Such as int: interger and char: character.
Data Structure The conceptual shape or arrangement of data.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function. It can only be referenced within that function.
Global Variable Variables whose scopes are not restricted to a particular part of a program.
Source Program The program in its original form.
Object Program The translated version of a program.
Python Collection Data Structures:
  • List: Ordered, Changeable, and allow duplicate values. Created using brackets.
    thislist = ["apple","cherry","grape"]
    print(thislist)
  • Tuple: Ordered and unchangeable. Written with round brackets.
    thistuple = ("apple","cherry","kiwi")
    print(thistuple)
  • Set: Unordered, no duplicates, unchangeable, and unidexed. Written with curly brackets.
    thisset{"apple","kiwi","cherry"}
    print(thisset)
  • Dictionary: Used to store data values in key:value pairs. With curly brackets, keys and values.
    Ordered, changeable, and do not allow duplicates.
    thisdict{
    "brand":"Ford",
    "model":"Mustang",
    "year": 1964}
    print(thisdict)
